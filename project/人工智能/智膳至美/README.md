# 2017级项目实训成果展示 

## 《智膳至美》 -  人工智能

###  项目简介

通过深度学习和智能推荐技术，精准把控饮食热量摄入，构建健康膳食管理系统。系统主要包括食物热量识别、健康膳食（或商家）推荐、饮食状况管理等模块。智膳至美让健康膳食带给您健康身体。
